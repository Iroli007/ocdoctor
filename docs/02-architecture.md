# 架构文档

## 技术栈

- **后端**: Python + FastAPI
- **数据库**: SQLite + SQLAlchemy
- **前端**: 待定 (建议 Next.js 或 Vue)
- **依赖管理**: uv
- **OCR**: 可替换抽象层 (PaddleOCR)
- **AI**: 统一 LLM 服务 (支持 OpenAI/Anthropic)

## 系统架构

## 模块职责

### API 层
- 只负责 HTTP 入参、调用 service、返回 response

### Services 层
- OCR 提取
- LLM 调用
- 卡片生成
- 比较题生成
- 小测题生成
- 复习逻辑

### Models 层
- 数据库实体定义

### Schemas 层
- 输入输出结构定义
